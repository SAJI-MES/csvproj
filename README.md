# Это простой СУБД для CSV/TSV

## О проекте
Проект на C# и позволяет работать с таблицами CSV/TSV формата.  

Особенности проекта:
1) Регистронезависимый выбор таблиц
2) Автоматическое определение разделителя (`;` для CSV, `\t` для TSV)
3) Построчное чтение файлов (не загружает весь файл в память)
4) Простая пагинация (skip / take)
5) Выбор конкретных столбцов (`*` для всех)

## Функции
1) Просмотр списка таблиц в папке
2) Просмотр списка столбцов выбранной таблицы
3) Пагинация данных
4) Выбор столбцов для вывода
5) Построчное чтение данных из CSV/TSV
